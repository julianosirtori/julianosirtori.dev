## O que é styled-components?

Styled Components é uma biblioteca `css-in-js`, isto é, ele permite escrever CSS dentro do javascript, transformando nossos estilos em 
componentes React. Segue um exemplo abaixo:

```js
const Button = styled.button`
  background: transparent;
  border-radius: 3px;
  border: 2px solid #BF4F74;
  color: #BF4F74;
  margin: 0 1em;
  padding: 0.25em 1em;
`
```

Essa variavel `Button` é um component react que podemos utilizar desta maneira: `<Button>Salvar</Button>`. O styled-components também 
permite que passamos parametros, desta forma o estilo pode mudar de acordo com nossas variaveis, isso tudo em tempo de execução, sem precisar 
escrever multiplas classes, deixando nosso css muito mais enxuto. 

```js
const Button = styled.button`
  color: ${({disabled}) => disabled ? #000 : #fff};
  background: ${({disabled}) => disabled ? #f0f0f0 : #BF4F74};
  margin: 0 1em;
  padding: 0.25em 1em;
`
```

Desta forma, podemos chamar o nosso component passando uma propriedade `disabled`, ficando assim `<Button disabled={true}>Salvar</Button>`

Para ter mais exemplos, e saber como usar o styled-components, você pode acessar diretamente a [documentação](https://styled-components.com/).
